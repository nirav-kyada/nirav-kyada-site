1: S-OFF your HTC wildfire s with XTC-Clip

2: Root your HTC with the method mentioned here at xda forums..

3: Partition your memory card (i used the FAT32 + ext2 )

4: update the necessary applications like... Google maps, youtube, etc..

5: install necessary softs ( it depends on you.. u are free...  ):
I used the
--> ES Explorer with root permision...
--> Script manager(free version) its use I have mentioned below...

6: install the app : Link2SD run the app and select the second partition type you used... then reboot

if u got it run without any problem after rebooting... thats it for you guys...

but if you been asked again select second partition type.. etc.. etc... ( the problem i had... )

then...

7: run the script manager...

menu>more>advance options> config > select browse as root

give the root permission to it...

now select the file

/system/etc/install-recovery.sh

select both > run as root
> run at boot

then save it... and reboot the phone... (u also can run it there also.. :P but i recommend to reboot.. )

after reboot and start the Link2sd... you are done....

you are able to use the great app... Link2SD.... thnx bro...

and when you restart the phone you will notice the linked apps are not present in the application...

to solve this >> run Link2SD > menu > more > quick reboot...

you will get your linked apps back...
